This is an example of how to use GPGPU to run process on graphics cards and not on CPU.

#### Install the python packages of Pipfile on virtual enviroment:
```
$ pipenv install
```
#### Start Virtual Enviroment:
```
$ pipenv shell
```
#### Run this project:
```
$ pipenv run main2.py
```
